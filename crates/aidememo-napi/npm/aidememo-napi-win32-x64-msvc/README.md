# aidememo-napi-win32-x64-msvc

Prebuilt native binary used by [`aidememo-napi`](https://www.npmjs.com/package/aidememo-napi)
on 64-bit Windows.

| Property | Value |
|---|---|
| Operating system | Windows (`win32`) |
| CPU | `x64` |
| Toolchain | MSVC |
| Node.js | 16 or newer |
| Binary | `aidememo-napi.win32-x64-msvc.node` |

Do not install this package directly. Install `aidememo-napi`; npm will select
this optional dependency automatically for a compatible machine.

```bash
npm install aidememo-napi
```
