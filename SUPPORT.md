# Support

## Documentation

Start with the [AideMemo documentation](https://aidememo.taeyun.me/docs/intro),
especially the installation, coding-agent integration, and operations guides.
Search existing issues and discussions before opening a new thread.

## Questions

Use [GitHub Discussions](https://github.com/taeyun16/aidememo/discussions) for
installation help, usage questions, integration design, and ideas that are not
yet actionable feature requests.

## Bugs and feature requests

Use the structured [issue forms](https://github.com/taeyun16/aidememo/issues/new/choose).
Bug reports should include a minimal reproduction and a redacted environment
description. Maintainers provide support on a best-effort basis and cannot
guarantee response or resolution times.

## Security

Do not disclose vulnerabilities in issues or discussions. Follow the private
reporting process in [SECURITY.md](SECURITY.md).
